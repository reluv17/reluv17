## 📂 Featured Projects

### 🔌 [Electricity Demand Forecasting](https://github.com/reluv17/electricity-demand-forecasting)
Hourly electricity consumption forecasting on 17 years of building energy data using Random Forest and Gradient Boosting. Achieved **Test R² = 0.9795** with near-zero overfitting gap. Features lag engineering, cyclical encoding, and TimeSeriesSplit cross-validation.  
`Python` `scikit-learn` `Time Series` `Ensemble Models` `GridSearchCV`

---

### 🏠 [Thermostat Override Prediction](https://github.com/reluv17/thermostat-override-prediction)
Binary classification predicting smart thermostat override events from one year of five-minute interval household data. Compared Logistic Regression (AUC = 0.69) vs Random Forest (**AUC = 0.9984**, Recall = 99.1%). Includes production deployment analysis and precision-recall trade-off discussion.  
`Python` `scikit-learn` `Classification` `Imbalanced Classes` `Feature Engineering`

---

### 🌡️ [Ecobee User Clustering](https://github.com/reluv17/ecobee-user-clustering)
Unsupervised segmentation of 1,000 Ecobee thermostat users from 8.5M hourly records. Applied K-Means, GMM, and DBSCAN to identify three behavioral personas — Energy Savers, Active Controllers, and Balanced Users — with product and marketing recommendations for each.  
`Python` `scikit-learn` `K-Means` `GMM` `DBSCAN` `Anomaly Detection`

---

### 🏡 [Ames Housing Regression](https://github.com/reluv17/ames-housing-regression)
House price prediction on 2,930 properties using Linear Regression, Ridge, and Lasso. Full pipeline from missing value handling and one-hot encoding through multicollinearity analysis (VIF), regularization, and GridSearchCV. Best model: **Lasso (Test R² = 0.8987)**.  
`Python` `scikit-learn` `Regression` `Regularization` `Feature Engineering`

---

### 💬 [ULMFiT Yelp Sentiment Classification](https://github.com/reluv17/ulmfit-yelp-sentiment)
Sentiment classification on Yelp reviews comparing fine-tuned AWD_LSTM (ULMFiT) against TF-IDF Naive Bayes. ULMFiT achieved **94.2% accuracy** vs 90.2% for the baseline, with stronger negative sentiment recall — critical for customer experience applications.  
`Python` `fastai` `NLP` `Transfer Learning` `Text Classification`

---

### 🖼️ [ResNet-18 Image Classification](https://github.com/reluv17/resnet18-stl10-transfer-learning)
Transfer learning with ResNet-18 fine-tuned on STL-10 across 10 image classes. Two-phase training strategy (frozen head → full fine-tuning) achieved **95.6% test accuracy**. Includes overtraining analysis, confusion matrix, and per-class performance breakdown.  
`Python` `PyTorch` `Computer Vision` `Transfer Learning` `CNN`
